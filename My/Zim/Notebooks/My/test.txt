Проведем тест 1:
Есть иерархия классов [1..N], есть некий класс K∈[1..N] и есть некий P

__Выяснить какие переопределенные методы будут вызываться каждым объектом в иерархии наследования (ссылка типа объекта, ссылка на 1й класс объекта):__
__1) делаем линейку из 5ти классов, __
__2) в 1м, 3м и 5м переопределим метод.__
__3) по линейке делаем вызовы.__

**т1) Для созданного объекта вызывается последнее для него переопределение метода, неважно какого типа допустимая ссылка (ее тип - от предка 1 до класса объекта).**

__Выяснить к каким переопределенным полям обращается метод в иерархии.__
__1)-2)-3) такие же.__

**т2) Для созданного в иерархии наследования объекта, переопределенный в иерархии метод обращается к той версии переопределенных полей, которая актуальна для вызванного метода, неважно какого типа ссылка.**

**т3 обобщающее) если в иерархии классов [1..N] присутствует переопределение члена (метода либо поля), то для объекта класса K∈[1..N]:**
**- будет вызван ближайший в сторону предков метод,**
**- и этод метод будет оперировать с ближайшими в сторону предков полями.**
