**Assignment - присваивание "=":**

Операция присваивания - это единственная операция, которая выполняется справа налево (сначала вычисляется то, что справа, результат присваивается тому что слева). Отсюда логичная фишка (как минимум у джавки) - можно присваивать цепочкой:
		a = b = c = d = 5; // всем по 5
