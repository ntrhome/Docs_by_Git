====== Overload ======
Created Среда 26 июня 2019

В джавке в классе можно объявить более одного метода с одним именем, при условии, что их сигнатуры различны - **перегрузка метода, полиморфизьм**. Различие должно быть в типе и/или количестве параметров.

Возвращаемые типы членов перегружаемого метода могут отличаться, но к перегрузке метода не ведут.

Хотя автоматическое преобразование типов в Java выпол­няется только в случае, если не обнаружено полное соответствие (т.е. если аргумент типа int, и есть метод с параметром именно int, то он и будет задействован, если же нет - то джавка пытается найти перегрузку продвигая аргумент), всеже следует учитывать влияние на перегрузку автоматического преобразования типов в джавке - расширяющее приведение ("продвижение") см. Primitives kkk:Primitives:Customisation.

Про перегрузку методов с переменным числом аргументов см в [[Methods:Varargs]].
