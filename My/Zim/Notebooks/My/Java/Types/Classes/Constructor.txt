====== Constructor ======
Created Вторник 25 июня 2019

**Конструктор:**
Конструктор инициализирует объект во время его создания.

Это спецметод, имя совпадает с именем класса, без возвращаемого значения.

Может перегружаться по общим правилам метода.

Не может быть перекрыт (какова бы ни была иерархия наследования класса от 1 до n, при создании объекта уровня k<=n будут вызваны поочередно все кострукторы, начиная с 1-го до k-го).

Обязательно запускается в самом начале строительства объекта. Причем внимательно: в момент его запуска члены уже будут ему известны, но поля еще не будут инициализированны (если таковые инициализирующие значения прописанны. Поэтому, для возможности корректного использользования полей в конструкторе, их инициализацию лучше выполнять в самом конструкторе).

Если пользователем прописан хоть один конструктор, то язык пытается найти соответсвующий заданным параметрам конструктор среди прописанных пользователем, при неудаче - ошибка.
Если пользователем не прописан ни один конструктор, то объект может быть создан только конструктором без параметров, причем такой конструктор будет создан незримо самим языком.

- конструктор дефаултный:
		Если никакой конструктор не прописан вручную пользователем, то языком будет создан незримый дефаултный без параметров.
- конструктор без параметров:
		Явно заданный пользователем конструктор без параметров, заменяет дефаултный.
- конструктор параметризированный:
		Принимает. Вох x = new Box(lO, 20, 15);
- констуктор перегружаем (см. [[Methods:Overload]]) нет особых требований к перегружаемому конструктору как к перегружаемому методу - т.е. если задан только один конструктор, но с тремя параметрами, значит объект создается только его вызовом, если есть еще один с пятью, значит один из двух.
