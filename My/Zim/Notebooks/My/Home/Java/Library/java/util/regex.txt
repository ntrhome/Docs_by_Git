====== regex ======
Created Saturday 10 August 2019

**Module java.base**
**Package **[[https://docs.oracle.com/en/java/javase/12/docs/api/java.base/java/util/regex/package-summary.html|java.util.regex]]

Classes for matching character sequences against patterns specified by regular expressions.

Consists of three classes: Pattern, Matcher, and PatternSyntaxException (etc).
