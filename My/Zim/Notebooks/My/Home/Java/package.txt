====== package ======
Created Wednesday 17 July 2019

Пакеты являются контейнерами классов. 

Они служат для разделения пространств имен классов. Т.е. одноименные классы могут мирно сосуществовать в разных пакетах.

Пакет имеют древовидную структуру.

Пакет может быть импортирован, присоеденяясь, таким образом, ветвью к основному пакетному дереву кода.

__Как поясняется в главе 16, пакеты играют также очень важную роль в модулях - новом языковом средстве, внедренном в версии JDK 9.__
