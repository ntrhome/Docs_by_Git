====== Boolean ======
Created Понедельник 17 июня 2019

**В Java нет числового эквивалента булевого значения **(т.е. числа не конвертируются в true или false, и true и false не конвертируются в числа).

Т.е. такие С-шные дела (что не 0 то True) в Java НЕ ПРОХОДЯТ:
	//if(5) ...
	//if(!0) ...
не работает и явный кастом:
	//if((boolean)0) ...
надо только явные булевые операции:
	if(5 != 0) ...

**Булевые операторы:**
-  "обычные" сравнения:
		&	- И
		|	- ИЛИ
		^	- искл. ИЛИ
		!	- НЕ
		==	- РАВНЫ?
		!=	- НЕ РАВНЫ?
- те же, но "составные с присваиванием": &=, |=, ^=.
- "короткие" сравнения (ограничивается вычислением только левого выражения, и только если этого недостаточно, то вычисляет правый - т.е. С-шный вариант):
		&&	- короткрое И
		||	- короткое ИЛИ
- тернальный выбор:
		?:	- условие ? результат-если-True : результат-если-False
