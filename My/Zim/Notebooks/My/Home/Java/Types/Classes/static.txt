====== static ======
Created Четверг 27 июня 2019

Статический член класса:
	- статическим может быть как метод, так и переменная,
	- существует вне объектов, в единственном для класса экземпляре,
	- по существу, является гло­бальным,
	- доступ извне - через имяКласса.статическийМембер;
в то же время, им присущи такие ограничения:
	- обращается непосредственно только к другим статическим членам,
	- обра­щаетсяться к нестатическим членам своего и чужого класса посредством объекта,
	- не применяют this и super.

Именно так в Java реализованы управляемые версии глобальных методов и переменных (?).

- - - - - - - - - - - - - **static-блок**

Если для инициализации статических переменных требуется произвести какие-то действия, то для этого достаточно объявить статический блок, который выполнится один раз при первой загрузке класса (после инициализации статических переменных при их объявлении, и до вызова статического main, если он есть и актуален):

class C {
		static int a =3;
		static float b;
		...
		static {
				b = 3.14 * a;
				System.out.println("Статический блок инициализирован.");
		}
		...
}
