====== Object ======
Created середа 10 липень 2019

**Класс Object** - встроенный  в джавку суперкласс, единственный в своем роде, т.к. от него неявно, но обязательно, ведут первое наследование абсолютно все любые классы.

Т.е. все остальные классы являются подклассами этого класса, и это оз­начает, что:
- через ссылочную переменную типа класс Object мы можем в джавке сылаться на вообще любой объект;
- поскольку в классе Object определен набор членов (хренова туча, которые иногда полезно переопределить), то они доступны любому объекту (некоторые методы являются final):
	- - **Object clone()** - "поверхностное" клонирование - создать новый объект, не отличающийся от клонируемого;
	- - **boolean equals(Object object)** - сравнивает ссылки (эквив obj == Other), следует перегружать на сравнение по содержимому.
	- - **final getClass().getName()** - возвращает строку вида "java.lang.Object" - пакет/имя класса.
	- - **toString()** - возвращает строку вида "java.lang.Object@372f7a8d" - пакет/имя класса + дескриптор.
	- - **void finalize() **устарел - вызывается перед удалением неиспользуемого объекта (не рекомендован с JDK 9).
	- - **Class<?> getClass()** - обобщение - получает класс объекта во время выполнения.
	- - **int hashCode()** - возвращает хеш-код, связанный с вызывающим объектом.
	- - **final void notify()** - возобновляет исполнение потока, ожидающего вызывающего объекта.
	- - **final void notifyAll()** - возобновляет исполнение всех потоков, ожидающих вызывающий объект.
	- - **String toString()** - Возвращает символьную строку, описывающую объект (перегрузить).
	- - **final void wait()** - ожидает другого потока исполнения.
	- - **final void wait(long миллисекунд)** - ожидает другого потока исполнения.
	- - **final void wait(long миллисекунд, int наносекунд)** - ожидает другого потока исполнения.
