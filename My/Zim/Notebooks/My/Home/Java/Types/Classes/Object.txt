====== Object ======
Created середа 10 липень 2019

Встроенный метод, от него неявно наследуются все классы джавки.

Имеет набор своих членов (хренова туча):
	- obj.equals(objOther) - сравнивает ссылки (эквив obj == Other), в своем классе перегружаем на сравнение по содержимому.
	- obj.getClass().getName() - возвращает строку вида "java.lang.Object" - пакет/имя класса.
	- obj.toString() - возвращает строку вида "java.lang.Object@372f7a8d" - пакет/имя класса + дескриптор.
