====== this ======
Created Вторник 25 июня 2019

Можно в текущем объекте пользовать "this." для разрешения неоднозначностей - гарантированный доступ к члену текущего объекта в условиях возможного перекрытия имен:
		this.width = w;

Но чаще всего, this передается или принимается в или из другого объекта для предоставления или получения возможности манипулирования со своими или чужими public-членами (взаимные вызовы, - игровые юниты, фабрика объектов и т.д.).
		class A{
					public int width;
					public void f(A obj){ obj.with=with;}
					...
		}

Передавая объект в аргументе, мы на самом деле передаем его ссылку this.
