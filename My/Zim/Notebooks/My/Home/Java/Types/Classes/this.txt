====== this ======
Created Вторник 25 июня 2019

Ссылка this ссылается на текущий объект.

Можно пользовать this для разрешения неоднозначностей - гарантированный доступ к члену текущего объекта в условиях возможного переопределения имен. Например, частый случай, когда безо всякой неоднозначности имена параметров метода скрывают поля класса:
		class A{
				int length;
				void f(int length){
						this.length = length;
				}
		}

Но чаще всего this передается в другой объект в механизме обратного вызова (игровые юниты, фабрика объектов и т.д.).

Му - Кстати, передавая объект в аргументе, мы на самом деле передаем его ссылку this.
