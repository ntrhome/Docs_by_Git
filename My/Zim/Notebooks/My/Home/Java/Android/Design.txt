====== Design ======
Created Понедельник 01 июля 2019

= = = = = = = = = = = = = = = = = = = = = 
Идея - создать тему "прилетающий дизайн" элементы через анимацию слетаются в 3Д интерфейс - проекция под небольшим углом к пользователю (видно какой элемент над кем, тени), который затем чуть доворачивается, становясь обычным андроидовским - выбрать библиотеку попроще, умеющую андроид-объекты с учетом оформления рисовать в 3Д. Короче: прога берет интерфейс другой проги, встраиваясь в процесс построения интерфейса, перехватывает чего и какие объекты, малюет полеты в 3Д, потом доворачивает и сдает на обслуживание системе.
