====== Components ======
Created вівторок 02 липень 2019

**Компоненты приложения:**
Приложение строится из компонентов, каждый из которых является точкой входа системы в приложение (может быть запущен другим приложением), но не все они являются точками входа для пользователя. В отличие от приложений для большинства других систем, в приложениях для Android отсутствует единая точка входа (нет функции main()). Некоторые из компонентов могут зависить друг от друга (Му - т.е. сами вызовут другие компоненты?).

Intent (намерение) - асинхронное сообщение (see [[+Activating:Intent]]).
